# 邮件管理系统Web程序设计 python285

## 项目概述

本项目是一个基于 Flask 框架、使用 Python 编程语言以及 MySQL 数据库的邮件管理系统Web程序。该系统旨在为用户提供一个方便、高效的邮件管理解决方案。

## 技术栈

- 前端：HTML/CSS/JavaScript
- 后端：Python + Flask
- 数据库：MySQL

## 功能模块

### 用户管理

- 用户注册
- 管理员登录与用户登录

### 邮件管理

- 远程管理个人邮件

### 系统管理

- 管理登录过的用户
- 管理邮件IP地址池
- 自主编辑菜单

## 系统角色

- 普通用户：可注册、登录、管理自己的邮件
- 管理员：除了普通用户权限外，还可以管理用户和邮件IP地址池，编辑菜单

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x

## 部署步骤

1. 安装Python和相关依赖
2. 安装MySQL数据库
3. 导入数据库结构
4. 运行Flask应用程序
